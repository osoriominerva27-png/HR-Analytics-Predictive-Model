# HR Analytics: Predicción de Rotación de Talento

## Descripción del Proyecto
Este proyecto tiene como objetivo analizar y predecir la rotación de empleados en una organización utilizando técnicas de Ciencia de Datos y Machine Learning. El propósito es proporcionar al área de Capital Humano una herramienta basada en datos para identificar proactivamente a los empleados en riesgo de abandono y fundamentar estrategias de retención.

## Tecnologías Utilizadas
* **Lenguaje:** Python (Pandas, NumPy, Scikit-Learn)
* **Visualización:** Seaborn, Matplotlib, Power BI
* **Modelado:** Random Forest Classifier

## Metodología
1. **Limpieza y Preparación:** Tratamiento de variables categóricas, eliminación de ruido y estructuración del dataset.
2. **Análisis Exploratorio (EDA):** Identificación de correlaciones clave entre variables como nivel salarial, satisfacción laboral y antigüedad.
3. **Modelado:** Implementación de un modelo de Random Forest para la clasificación de deserción, evaluando su desempeño mediante métricas como Recall y Matriz de Confusión.

## Hallazgos Principales
Se determinó que los factores principales que influyen en la rotación son: MonthlyIncome, OverTime_Yes y Age.
